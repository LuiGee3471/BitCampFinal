# Teaming
오프라인 컴퓨터 학원의 정보 공유·관리 플랫폼으로 학생은 자신의 문제·자료·영상 등을 다른 학생들과 공유할 수 있습니다. 또한, 프로젝트 팀에게 일정 관리, 의견 교환, 문제 공유의 기능을 제공합니다. 강사는 담당하는 학생의 활동을 관리할 수 있고 자신이 사용한 강의 자료를 재활용할 수 있습니다.
